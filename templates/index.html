Quick Run

On-demand ranking for a specific batch or college

Batch Mode

Rank all colleges in a specific group. Groups are formed by sorting Source by Add column (descending), then splitting into groups of 50.

Example: Batch 1 = first 50 colleges, Batch 2 = next 50, etc.
College Mode

Rank one specific college by its College ID. The first course for that college is picked automatically — consistent with the main pipeline.

Example: College ID = 12345 (from Source sheet column A)

Data Management

How to add, manage, and remove college data

⚠️ Important Rules — Read Before Modifying Data
🚫
Do NOT delete the Checkpoint column The pipeline uses this to track the 15-day ranking cycle. Deleting it resets all tracking.
🚫
Do NOT modify the Intermediate sheet manually This sheet is managed entirely by the pipeline. Any manual changes will be overwritten.
🚫
Do NOT change Source sheet column order Columns are read by position. Moving or renaming columns breaks data ingestion.
🚫
Do NOT delete the Final or Content tabs These hold your historical ranking data. Deleting them loses all previous run data.
✅
You CAN add new rows to the Source sheet New colleges are automatically detected and queued for ranking on the next pipeline run.
✅
You CAN edit college_name and course_name Display names in Source can be updated at any time — the pipeline uses college_id for tracking.
📥 How to Add New Colleges to Source Sheet
  • 1
    Open the Google Spreadsheet Click the link in the header above, or go to your spreadsheet.
  • 2
    Go to the "Source" tab Click the "Source" sheet tab at the bottom of the spreadsheet.
  • 3
    Add a new row with the required columns Fill in: college_id, course_id, college_name, course_name, keyword, Add (the date/priority field for selection ordering).
  • 4
    Wait for the next auto-run (midnight) or trigger manually New colleges are auto-detected. Run the pipeline manually from the "Run Pipeline" page if you don't want to wait.
  • 5
    Verify in Rankings After the pipeline runs, check the "Rankings" page to confirm your new colleges appear in the Final sheet.
🗑️ How to Remove a College
Removing a college from Source will stop it from being ranked in future runs. However, existing data in Intermediate and Final sheets is NOT automatically deleted.
  • 1
    Delete or comment out the row in Source sheet Either delete the entire row, or add a prefix like REMOVED_ to the college_id to deactivate it.
  • 2
    Optionally clean Intermediate sheet If you want to remove historical keyword data, you can manually delete the corresponding rows in the Intermediate sheet — but this is optional.
